# Adult-Income-Classification
Statistical analysis on the UCI dataset "adult income classification", meant to predict whether someone earns more or less than 50K$/year based on education, age and other factors. 

In this file you will find an exploratory analysis of the dataset, preprocessing, data augmentation, and the training and validation of many machine learning models for classification tasks, e.g. Random Forest Classifier, Support Vector Classifier, K-nearest neighbours classifier, and more.
